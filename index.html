Zookeeper Helm Chart

Overview

Zookeeper is a centralized service for maintaining configuration information, naming, and providing distributed synchronization. This Helm chart deploys Zookeeper into your Kubernetes cluster.

Get Repository Info

helm repo add zookeeper https://leozw.github.io/zookeeper/

Install Chart

helm install zookeeper zookeeper/zookeeper -n kafka --create-namespace
